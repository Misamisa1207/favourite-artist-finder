Project title: Favourite Artist Finder

Brief project description: 
This is a React Native app that helps users search for their favourite artists by using Deezer API.
This also helps users to find related artist of their favourite artists.

Setup instructions:
-Backend-
npm install
npm run dev

-mobile-
npm install
npx expo start
-> scan QR code by Expo Go

.env
EXPO_PUBLIC_API_URL=http://****YOUR_LOCAL_IP***:5000

API used
Deezer API
-for search artists
-get artist details
-get related artists

Features implemented
-search artists by name
-show artist information
-show related artists
-reusable ArtiscCard component
-built 3 screens
-useState, useEffect, onPress, TextInput, router, params,
 FlatList, ActivityIndicator, Pressable, TouchableOpacity, StyleSheet
